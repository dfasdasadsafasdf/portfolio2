# Arduino Safety Monitoring System

Этот проект — одна из моих первых практических работ с микроконтроллером Arduino.
Целью проекта было изучение базовых принципов работы встроенных систем и датчиков, а также построение простой системы мониторинга безопасности.

Система отслеживает несколько параметров окружающей среды и реагирует на потенциально опасные условия.

Микроконтроллер Arduino постоянно считывает данные с подключённых датчиков.

Данные с датчиков анализируются в реальном времени.

Если значения превышают заданные пороговые значения — система определяет это как опасную ситуацию.

В случае обнаружения опасности система:
включает светодиодный сигнал
активирует звуковой сигнал

Так же я сделал ручную тревогу, ВКЛ/ВЫКЛ, отслеживание датчиков.
